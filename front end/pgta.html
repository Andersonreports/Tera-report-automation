User Guide

Overview

The PGT-A Report Generator creates professional Preimplantation Genetic Testing for Aneuploidies (PGT-A) reports in PDF format.

Manual Entry

  1. Fill in Patient Information (Patient Name and Sample Number are required).
  2. Set the Number of Embryos and click Update.
  3. For each embryo, fill in the Summary, Interpretation, and Chromosome Statuses (1-22).
  4. Optionally upload a CNV Chart Image (PNG/JPG) for each embryo.
  5. Use Copy Last Embryo to duplicate the previous embryo's data.
  6. Click Preview to see the report, or Generate Report(s) to produce the final PDF.

Bulk Upload

  1. Download the Excel Template.
  2. Fill in one row per embryo. Patients are grouped by Sample_Number.
  3. Upload the completed file and click Generate All Reports.

Report Comparison

Upload two PDF reports (manual and automated) to compare them. Differences in text, patient data, and embryo results are highlighted.
