PGT-A Report Generator — User Guide

Pre-Implantation Genetic Testing for Aneuploidies

1. Manual Report Entry

  • Patient Information: Fill all 14 patient fields. Use DD-MM-YYYY for all dates.
  • Page 1 — Results Summary: Set the number of embryos, then for each row enter Embryo ID, Result Summary (color-coded), Interpretation (color-coded), and MTcopy.
  • Page 4+ — Embryo Details: Per embryo: Result Description, Autosomes, Sex Chromosomes, CNV chart upload, Inconclusive Comment, and all 22 chromosome statuses with mosaic %.
  • Color Coding: Red = Abnormal/Aneuploid/Multiple. Blue = Mosaic. Black = Normal/Euploid.
  • Copy Last Embryo: Increments count and copies all data from the last embryo.
  • Live Preview: Auto-updates 1.2 seconds after you stop typing. Click Refresh Preview to force update.
  • Save Draft / Load Draft: Saves work as a named cloud draft; reload from any browser session.
  • Generate Report: Creates a downloadable PDF uploaded to Supabase storage.

2. Bulk Upload & Batch Processing

  • Download Template: Use the button to get the correct Excel template with all required columns.
  • Import: Upload your Excel (.xlsx). Rows with the same Sample_Number are grouped as one patient's embryos.
  • Review: Click a patient in the list to load their data into the middle editor panel.
  • Edit: Modify patient info and embryo data; live preview updates automatically.
  • CNV Charts: Upload CNV images per embryo directly in the editor.
  • Generate Selected: Generates reports for checked patients. Generate All: Processes all patients.
  • Save Bulk Draft: Preserves all edited bulk data for later continuation.

3. Report Comparison

  • Upload two PDFs — auto-generated (left) and reference (right).
  • Click Compare PDFs to render both side by side.
  • Use Prev / Next Page to navigate through report pages.
  • The Auto Diff Summary shows text differences extracted from both PDFs.
  • Click Export Diff as HTML to save the diff report for documentation.

4. Color Coding System

  • Red: CNV status G/L/SG/SL. Aneuploid, Chaotic embryo, Multiple chromosomal abnormalities result.
  • Blue: CNV status M/MG/ML/SMG/SML and combined states. Low/High/Complex level mosaic, Mosaic chromosome complement.
  • Black: CNV status N (Normal). Normal chromosome complement, Euploid, Inconclusive, Low DNA.

5. Excel File Format (Bulk Upload)

  • Patient columns: Patient_Name, Spouse_Name, PIN, Age, Sample_Number, Referring_Clinician, Biopsy_Date, Hospital_Clinic, Sample_Collection_Date, Specimen, Sample_Receipt_Date, Biopsy_Performed_By, Report_Date, Indication
  • Per-embryo columns: Embryo_ID, Result_Summary, Result_Description, Autosomes, Sex_Chromosomes, Interpretation, MTcopy
  • Rows with the same Sample_Number are grouped as one patient's multiple embryos.
  • Use the Download Template button for the correct format with all column headers pre-filled.

6. Chromosome Status Codes

  • N — Normal (black)
  • G / L — Gain / Loss, non-mosaic (red)
  • SG / SL — Single chromosome gain/loss (red)
  • M — Mosaic (blue)
  • MG / ML — Mosaic gain/loss (blue)
  • SMG / SML — Single mosaic gain/loss (blue)
  • SL/SG, SG/SL, SML/SMG, SMG/SML — Combined mosaic states (blue)

7. Productivity Tips

  • Copy Last Embryo: Use this to quickly duplicate embryo data when multiple embryos have similar results.
  • Collapse/Expand: Click any embryo header to collapse its details — useful when managing many embryos.
  • Drafts: Drafts are stored in the cloud — load from any device or browser.
  • Results Summary Comment: Optional free-text that appears below the page 1 results table.
  • Inconclusive Comment: Appears beneath the CNV chart on page 4+ for inconclusive results.

8. Quick Start

  • Manual: Fill patient details → Set embryo count → Fill Page 1 summary rows → Fill Page 4+ embryo details → Generate Report
  • Bulk: Download template → Fill data → Upload Excel → Select/edit patients → Generate Selected or Generate All
  • Comparison: Upload two PDFs → Click Compare PDFs → Navigate pages → Export diff if needed
  • Use Save Draft at any point to preserve your work in progress.
Pro Tip: The live preview updates automatically while you type — no need to click Refresh every time. Use the search box in Bulk Upload to quickly find patients by name in large batches.
