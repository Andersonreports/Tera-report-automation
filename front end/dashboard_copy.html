TERA Report Generator — User Guide

Transcriptome based Endometrial Receptivity Assessment

1. Manual Entry

  • Fill in the form fields. The live preview updates automatically after a short pause.
  • Click Refresh Preview to immediately render all report pages.
  • TERA Result: choose Receptive, Pre-receptive, or Post-receptive.
  • Biopsy Date & Time: enter YYYY-MM-DD HH:MM:SS or DD-MM-YYYY HH:MM.
  • P+ Hours: numeric hours since P4/hCG injection.
  • Time for Report: blastocyst hours + margin (example: 144 + 2).
  • Save Draft / Load Draft: stores the form as JSON for later reuse.
  • Set the branding option and click Generate Report.

2. Bulk Upload

  • Click the upload zone to load a .xls or .xlsx automation report.
  • All rows appear in the left patient list.
  • Select a row to edit it in the center editor panel.
  • The live preview updates in the right panel when you select a patient.
  • Use Generate Selected or Generate All to export.
  • Save Draft stores edited rows as JSON.

3. Report Comparison

  • Select two PDFs — auto-generated (left) and reference (right).
  • Click Compare PDFs to render both pages side by side.
  • Use Prev Page / Next Page to navigate through report pages.
  • The diff summary panel shows text-level differences automatically.

4. Result Types & Key Fields

  • Receptive: endometrium is within the window of implantation.
  • Pre-receptive: biopsy too early; window not yet open.
  • Post-receptive: biopsy after the window.
  • Time for Report example: 144 + 2
  • Python 3.10+, PyQt6 ≥ 6.6, ReportLab ≥ 4.0, pandas, openpyxl, Pillow.
Tip: Draft files are JSON and can be edited before loading. Save Draft stores your work locally and on the server.
