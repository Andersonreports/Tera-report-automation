TERA Report Generator - User Guide
Transcriptome based Endometrial Receptivity Assessment

1. Manual Entry

  • Fill in the form on the left. Fields auto-trigger a live PDF preview after a short pause.
  • Click Refresh Preview to immediately render all 3 report pages.
  • TERA Result: choose Receptive, Pre-receptive, or Post-receptive.
  • Biopsy Date & Time: enter YYYY-MM-DD HH:MM:SS or DD-MM-YYYY HH:MM.
  • P+ Hours: numeric hours since P4/hCG injection.
  • Time for Report: blastocyst hours + margin (example: 144 + 2).
  • Save Draft / Load Draft: stores the form as JSON for later reuse.
  • Set the output folder then click Generate Report.

2. Bulk Upload

  • Click Browse to load a .xls or .xlsx automation report.
  • All rows appear in the left patient table.
  • Select a row to edit it in the middle panel.
  • The preview updates automatically.
  • Use Generate Selected or Generate All.
  • Save Draft stores edited rows as JSON.

3. Report Comparison

  • Select two PDFs — auto-generated (left) and reference (right).
  • Click Compare PDFs to render both pages side by side.
  • Use Prev Page / Next Page to navigate pages.
  • Drag the center divider to resize panels.

4. Result Types & Key Fields

  • Receptive: endometrium is within the window of implantation.
  • Pre-receptive: biopsy too early; window not yet open.
  • Post-receptive: biopsy after the window.
  • Time for Report example: 144 + 2

5. System Requirements & Installation

  • Python 3.10+, PyQt6 ≥ 6.6, ReportLab ≥ 4.0
  • pandas, xlrd, openpyxl, Pillow, pypdfium2
  • Install dependencies: pip install -r requirements.txt
  • Install fonts: python install_fonts.py
Tip: The output folder is remembered between sessions. Draft files are JSON and can be edited before loading.
